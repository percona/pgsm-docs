## Release highlights

### Bug fix

[PG-2116](https://perconadev.atlassian.net/browse/PG-2116) - Fixed an issue where `pg_stat_monitor` could truncate queries in the middle of multi-byte characters when `pg_stat_monitor.pgsm_query_max_len` was reached, resulting in invalid byte sequence errors when querying `pg_stat_monitor`.
